MERN Contact Management App
A full-stack Contact Management application built using the MERN stack. This project is successfully deployed with the frontend on Netlify and the backend on Render.

Live Demo
Frontend URL: https://mern-contact-frontend.netlify.app
Backend API URL: https://mern-contact-app-bmh1.onrender.com

Tech Stack
Frontend: React.js
Backend: Node.js & Express.js
Database: MongoDB Atlas (Cloud)
Deployment: Netlify (Client) & Render (Server)

Project Structure
/client: Contains the React frontend source code.
/server: Contains the Node.js/Express backend API code.

Deployment Configuration
Backend (Render): Deployed using the server directory as the Root Directory.
Frontend (Netlify): Deployed using the client directory as the Base Directory.
Connection: The frontend communicates with the backend via the REACT_APP_API_URL environment variable.
